# 🗳️ Simple Voting System on Soroban

## 📌 Project Description

This project is a basic decentralized voting system built using **Soroban smart contracts** on the Stellar network. It allows users to create proposals and vote on them in a transparent and tamper-proof way.


## ⚙️ What It Does

* Users can create proposals (identified by a unique name)
* Anyone can vote on an existing proposal
* The contract keeps track of vote counts for each proposal
* Vote data is stored on-chain for transparency and immutability

## ✨ Features

* 🧱 Built on Soroban (Stellar smart contract platform)
* 📊 Simple and efficient vote counting
* 🔒 Immutable on-chain storage
* ⚡ Low-cost transactions
* 🪶 Lightweight contract design
